## EchoMind 智能客服系统

**技术栈**：Python、FastAPI、Anthropic Claude API、Redis、ChromaDB、Prometheus、Docker Compose

**项目描述**：基于 LLM 的企业级智能客服系统，支持三路融合意图识别实现多角色 Agent 路由（文档咨询、技术支持、工单处理），结合 RAG 知识库增强问答与三级记忆管理，实现多场景智能问答与上下文连续对话，并支持人工紧急事件标记与升级

**项目职责：**

- **三路融合意图识别与多角色路由**：设计 LLM + Embedding + Pattern Matching 三路分类体系，通过加权投票融合结果，根据用户意图动态路由至文档咨询、技术支持、工单处理等角色化 Agent；支持紧急事件标记与人工升级

- **RAG 混合检索优化**：构建查询改写 + BM25 + 向量检索多路召回体系，通过 RRF 融合排序缓解不同检索器分值差异；结合 Parent-Child Chunk 与 Auto-Merging 机制，缓解知识切分导致的上下文割裂问题，提升复杂问答场景下的答案完整性

- **三级记忆管理**：设计 Redis 工作记忆 + ChromaDB 历史摘要记忆 + 用户画像长期记忆架构，动态召回历史上下文增强多轮对话连续性；会话超过 15 轮自动触发 LLM 摘要压缩，降低上下文窗口占用与 Token 成本

- **可观测性与评测体系建设**：集成 Langfuse 对查询改写、召回、重排、生成等 RAG 全链路进行 Trace 监控，快速定位召回瓶颈；构建 LLM-as-Judge 自动评测体系，从相关性、准确性、完整性、有用性四个维度评估回答质量，支持历史回归测试与效果迭代

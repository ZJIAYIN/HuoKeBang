## EchoMind 智能客服系统

::: 后端开发
:::

- **技术栈**：Python、FastAPI、Anthropic Claude API、Redis、ChromaDB、Prometheus、Docker Compose

- **项目描述**：一个基于 LLM 的企业级智能客服系统，集成三路融合意图识别、多 Agent 路由编排、Redis + ChromaDB 三级记忆、RAG 知识库增强回复、LLM-as-Judge 端到端评测

**项目职责：**

- **三路融合意图识别**：设计 LLM + Embedding + 关键词三路并行分类器，加权投票合并，单路故障自动兜底降级，保证分类链路高可用

- **文档分割策略**：递归分割优先保留语义边界，token 兜底保证不丢内容，overlap 防止上下文断裂

- **RAG 混合检索 + 父子分块**：多路召回（查询改写 + 向量检索 + BM25）保证召回完整性，RRF 粗排消除分数量纲差异；二级分块 + Auto-merging 平衡检索精度和上下文完整性，合并后经 LLM 精排提升 Top-K 质量

- **三级记忆管理**：Redis 工作记忆 + ChromaDB 情景记忆/用户画像三级架构，超 15 条自动 LLM 摘要压缩，兼顾上下文连续性与 token 成本

- **RAG 全链路可观测 + LLM-as-Judge 双层评估**：集成 Langfuse 对 RAG 检索做全流程 trace（查询改写 → 多路召回 → 重排），定位召回瓶颈；LLM-as-Judge 对生成层四维评分（相关性、准确性、完整性、有用性），覆盖检索精度与生成质量，支持多轮评测和历史回归检测

## EchoMind 智能客服系统

**技术栈**：Python / FastAPI ｜ Milvus / Redis ｜ Claude API / Langfuse ｜ Docker Compose

**项目描述**：基于 LLM 的企业级智能客服系统，支持三路融合意图识别实现多角色 Agent 路由（文档咨询、技术支持、工单处理），结合 RAG 知识库增强问答与三级记忆管理，实现多场景智能问答与上下文连续对话，并支持人工紧急事件标记与升级

**项目职责：**

- **三路融合意图识别与多角色路由**：设计 LLM + Embedding + Pattern Matching 三路分类体系，通过加权投票融合结果，根据用户意图动态路由至文档咨询、技术支持、工单处理等角色化 Agent；支持紧急事件标记与人工升级

- **RAG 混合检索与在线质量拦截**：基于 Milvus 构建 Dense + Sparse 双向量混合检索，RRF 融合排序消除分数量纲差异；检索后 LLM-as-Judge 置信度评判，低分 query 自动拦截并持久化至 Redis 形成反馈闭环，指导知识库定向补全

- **三级记忆管理**：设计 Redis 工作记忆 + Milvus 历史摘要记忆 + 用户画像长期记忆三层架构，动态召回历史上下文增强多轮对话连续性；会话超过 15 轮自动触发 LLM 摘要压缩，降低上下文窗口占用与 Token 成本

- **检索评测与全链路可观测**：构建检索评测管线 `/eval/retrieval`，LLM-as-Judge 评判 Top-K chunk 与参考答案语义相关性量化 Recall@K，反哺检索策略迭代；集成 Langfuse 对查询改写、召回、重排全链路 Trace 监控，定位召回瓶颈
